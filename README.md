# ETL Training Portal MVP

Внутренний MVP-портал для загрузки Excel-файлов, проверки данных и записи в MSSQL.

## Возможности

- вход для сотрудников отдела обучения
- загрузка Excel (`.xlsx`, `.xls`)
- проверка структуры, обязательных полей, типов данных и дубликатов
- предпросмотр до импорта
- защита от повторной загрузки файла по SHA-256
- импорт в MSSQL
- журнал загрузок и ошибок

## Быстрый старт

1. Скопируйте `.env.example` в `.env`.
2. Установите зависимости: `npm install`
3. Выполните SQL из `database/schema.sql`
4. Запустите приложение: `npm run dev`

## Шаблон Excel

Первая строка файла должна содержать колонки:

- `employee_id`
- `full_name`
- `email`
- `course_code`
- `course_name`
- `completion_date`
- `score`

## Ограничения MVP

- пользователи задаются через `AUTH_USERS` в `.env`
- для предотвращения повторной загрузки используется хэш файла
- импорт идет в `dbo.training_records`
- перед продуктивным запуском стоит подключить AD/LDAP, HTTPS и вынести сессии в Redis/SQL
